  <h1 align="center">HTTP web server build from scratch in C++</h1>
</div>
In this C++ application, an HTTP web server has been developed entirely from scratch, without relying on any third-party libraries for HTTP webserver. The server utilizes a TCP/IP connection and is registered to an IP address on the computer through the use of a TCP/IP socket. The internet protocol has been defined using <a href="https://pubs.opengroup.org/onlinepubs/7908799/xns/syssocket.h.html">sys/socket.h</a>  and <a href="https://pubs.opengroup.org/onlinepubs/7908799/xns/arpainet.h.html">arpa/inet.h</a>.
